A- ¿Quién inventó el sistema de control de versión Git y por qué?
B- ¿A quién pertenece actualmente Github y por qué?
C- ¿Hay otra forma que no sea la terminal para trabajar con Github?

A- El sistema de control de versión Git fue inventado por Linus Torvalds en 2005 para ser utilizado en el desarrollo del kernel de Linux.
B- GitHub pertenece actualmente a Microsoft porque la compañía ha adoptado una estrategia nueva, mucho más abierta al mundo del software libre y a la integración con otras plataformas y servicios.
C- Si, la hay, a traves de Git GUI y su interfaz gráfica.
